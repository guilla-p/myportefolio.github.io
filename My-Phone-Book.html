Projet « My Phone Book »

Développement d’un annuaire d’entreprise grâce au framework Laravel contenant les entreprises partenaires et leur collaborateurs.

20 novembre 2021 Pierre GUILLAUME
  • LinkedIn
  • Email

L'objectif de ce projet a été de mettre en place grâce au framework Laravel, un annuaire contenant les différents collaborateurs des entreprises partenaires de l'entreprise pour laquelle l'annuaire est crée.

Les outils utilisés pour ce projet ont été :

  • framework : Laravel
  • langage : PHP
  • Gestionnaire Base de Données : MariaDB

L'objectif premier a été de créer les entités avec les données suivantes :

  • Entreprises
    • Nom de l'Entreprise
    • Rue
    • Code Postal
    • Ville
    • Numéro de téléphone
    • Email
  • Collaborateurs
    • Civilité
    • Nom
    • Prénom
    • Rue
    • Code Postal
    • Ville
    • Numéro de téléphone
    • Email
    • Entreprise (seul peut être choisi une entreprise déjà rentrée dans la base de donnée)

Ensuite, 3 profils utilisateurs devaient être mis en place avec pour chacun leurs droits spécifiques

  • Admin : Ajouter, Modifier et Supprimer
  • Gestionnaire : Ajouter et Modifier
  • Utilisateur : Consultation seule

Enfin, la mise en place d'une barre de recherche pour chacune des deux entités. La barre de recherche permet de faire une recherche selon les informatiosn suivantes:

  • nom/prénom
  • Entreprise
  • Numéro de téléphone
  • Email
  • LinkedIn
